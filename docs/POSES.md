# Pose Reference — Meccha Chameleon Trainer

## Shape > color

A perfect eyedropper job fails if your silhouette screams "humanoid."

## Pose catalog

| Pose | Best maps | Risk |
|:--|:--|:--|
| **Wall press** | Narrow corridors, flat partitions | Shoulder width vs object |
| **Crouch** | Furniture-heavy rooms, low props | Head above sofa line |
| **Lie flat** | Open floors, rugs, pool decks | Shadow length at angle |
| **Balloon curl** | Colorful party rooms, kid stages | Size mismatch |
| **Frame flat** | Gallery walls, poster rooms | Frame thickness |
| **Emote bend** | Cluttered prop piles | Animation lock discipline |

## Map-type cheat sheet

- **Cluttered indoor** → crouch or emote into prop silhouette  
- **Wide empty hall** → lie flat + aggressive floor sampling  
- **Vertical walls only** → wall press + minimal limb exposure  
- **Workshop meme maps** → read the joke object first, then pose *as* the joke  

## Freeze discipline

Once hide phase starts: **zero camera nudge**. Seekers track micro-jitter faster than bad paint.
